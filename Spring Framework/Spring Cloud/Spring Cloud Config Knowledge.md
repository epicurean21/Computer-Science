# Spring Cloud Config - Knowledge

## Spring Actuator

> Spring actuator란 스프링 부트에서 운영 중인 application을 HTTP 또는 JMX를 이용해서 모니터링하고 관리할 수 있는 기능을 제공하는 **라이브러리**. 
> 
> 웹의 상태 모니터링, metric, traffic 정보 그리고 db 상태 등을 알 수 있다.
